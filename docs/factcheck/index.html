Query Parameters

paramsvaluedescription
tagstringTo get factchecks of a tag
slugstringTo get factcheck by slug
categorystringTo get factchecks of a category
idObjectIdTo get factchecks by object id
claimantstringTo get factchecks of a particular claimant
limitnumberTo limit number of factchecks
nextstringTo get next factchecks
previousstringTo get previous factchecks
sortBystringTo sort by field
sortAscbooleanTo set sort by ascending

### Query Parameters

**params**|**value**|**description**
-----|-----|-----
tag | string | To get factchecks of a tag
slug | string | To get factcheck by slug
category | string | To get factchecks of a category 
id | ObjectId | To get factchecks by object id
claimant | string | To get factchecks of a particular claimant
limit | number | To limit number of factchecks
next | string |To get next factchecks
previous | string |To get previous factchecks
sortBy | string | To sort by field
sortAsc | boolean | To set sort by ascending
    
## Get factcheck

**<span style="background-color:green; color : white; font-size : 18px">`GET`</span>**  <span style="font-size : 18px">`/api/v1/factchecks/:key`</span>

**params**|**description**
-----|-----
key |  To get factcheck by Object Id or slug
